Waste Classification Using Deep Learning

Machine Learning Engineer (Individual Academic Project)

An image classification system that automatically categorizes waste into Organic and Recyclable classes using a pre-trained InceptionV3 deep learning model and transfer learning techniques.

report_problem The Challenge

Building an accurate image-based waste classification system using a limited dataset, while avoiding overfitting and ensuring the model can generalize well to unseen images with different lighting conditions and backgrounds.

check_circle The Solution

Implemented a transfer learning approach using the InceptionV3 pre-trained model. Performed image preprocessing and data augmentation, froze base layers, and fine-tuned the classification head. Model performance was optimized using callbacks such as EarlyStopping, ModelCheckpoint, and ReduceLROnPlateau.
